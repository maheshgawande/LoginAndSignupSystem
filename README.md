# LoginAndSignupSystem

Login and Signup system with CRUD functions using PHP. (Create, Read, Update, Delete)

# Requirements

- Web server (`XAMPP`, `WAMP`, etc.)
- Database with name `userdata`
- Table inside database with name `users`
